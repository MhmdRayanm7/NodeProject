# Light of Vision – Node.js Project

## Description
Light of Vision is a game-inspired data management system.
This project manages players, levels, items, and player progress
for the game "Light of Vision".

## Technologies
- Backend: Node.js + Express
- Database: MySQL
- Frontend: React
- Authentication: Login / Register

## Database Tables
- users
- levels
- items
- player_progress
- game_logs
